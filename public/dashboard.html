Setup Guide
Get your WhatsApp bot running in minutes
Step 1 - Install Dependencies

Make sure you have Node.js 16+ installed, then run:

npm install
Step 2 - Start the Server
node index.js

A QR code will appear in your terminal.

Step 3 - Link WhatsApp
  1. Open WhatsApp on your phone
  2. Tap the three dots (menu) and select Linked Devices
  3. Tap Link a Device
  4. Scan the QR code shown in the terminal
Step 4 - Open Dashboard

Once connected, open this dashboard at:

http://localhost:3000
Important Warning

This uses whatsapp-web.js, an unofficial library that automates WhatsApp Web. This is against WhatsApp Terms of Service. Use responsibly and at your own risk. Avoid sending spam or bulk messages. Your account could be temporarily or permanently banned.
